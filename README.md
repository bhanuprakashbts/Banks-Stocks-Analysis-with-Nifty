# Banks-Stocks-Analysis-with-Nifty

Analysis of three stocks in Banking Sector, namely State Bank of India(SBIN),Icici Bank(ICICIBANK) and HDFCBank(HDFCBANK) from National Stock Exchange of India.

Bank Stocks Analysis is the project on basic analysis and visualization using data fetched by NSEpy. 

By looking at data from the stock market, particularly some giant Banking stocks, visualized different aspects of it, and finally looked at returns and correlation of returns a stock, and beta calculationsbased on its previous one year performance history.
